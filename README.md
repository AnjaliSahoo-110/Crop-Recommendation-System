# Crop Recommendation System

Machine Learning based Crop Recommendation System using Random Forest.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
- 2200+ records
- 22 crop categories

## Model
- Random Forest Classifier

## Accuracy
- 99.32%

## Features
- Crop Prediction
- Feature Importance Analysis
- Correlation Heatmap
- Data Visualization
